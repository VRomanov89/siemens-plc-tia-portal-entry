# Section 2 — Setting Up TIA Portal: Conclusion

**Estimated segment duration:** 3–5 minutes
**Type:** Section recap (structural — not counted in lesson totals)

---

## What You Covered in This Section

Two more lessons done — and the setup headaches that trip up most beginners are now behind you.

**From Lesson 2.1:**
- TIA Portal versions are not interchangeable — match the software version to your PLC's firmware version before installing
- The 21-day trial includes STEP 7 Professional; for this course STEP 7 Basic capability is sufficient
- PLCSIM requires a separate export control registration — this is a compliance formality, not a technical error. If you haven't submitted the form yet, do it now
- TIA Portal needs capable hardware: 16GB RAM, SSD, and a physical Ethernet adapter for later hardware work

**From Lesson 2.2:**
- Install only what this course needs: STEP 7 and PLCSIM — skip WinCC Runtime and drive packages
- The TIA Portal interface is context-sensitive — what you see changes based on what's selected, not because the software is broken
- You can now locate the Project Tree, Device View, Network View, Program Editor, Tag Table, and Watch Table without hunting for them

- [ ] A blank project created and navigated successfully
